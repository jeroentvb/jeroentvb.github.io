Hoi! Ik ben Jeroen

Ik ben een student communication & multimedia design. Ik zit momenteel in mijn tweede jaar. Ik vind het leuk om websites te bouwen maar ook om defunctionaliteit ervan te ontwerpen. So basically a frontender with some affinity for UX/interaction. You may notice that I'm not a very good visual designer. I usually make things pretty when I have acces to a styleguide.

More about me

I like to tinker with computers, mainly software. An example may be that I use a so called hackintosh (non-apple computer that runs MacOS). That tinkering got me to learn HTML & CSS

In my spare time I like to go windsurfing or play games. I also really like to go skiing (although due to travel time and cost I don't do this often. Living in the Netherlands isn't ideal for this..). I also like to edit video of my ski trips and windsurf sessions. That way I thought myself how to edit in Vegas Pro.

These are some of my skills:

  • HTML & CSS/SASS (SCSS)
  • A bit of JS
  • Photoshop
  • Some InDesign
  • Video editing with Vegas Pro

Mijn werk

Project WEB - Stories website for the NS

This was a fictional project for the NS. It's a website on which travellers can find and read stories before, while and after travelling. The whole site is in Dutch.

The website is completely hand-made in vanilla HTML/CSS/JS (with a few JS library's). I learned a lot about semantic HTML and how to make a website responsive while working on this project.

Project documentation can be found here, it's all in Dutch.

Professor Layton puzzle in JavaScript

This is the project I made for 'inleiding programmeren' a subject at which I learned the basics of JavaScript. I really liked building this small game as I had never used JS before. Learning JS really was an eye opener as to what can be done with a static HTML/CSS/JS page.

Note: the app has sound and doesn't scale well on mobile devices.

SNOW edits

In my free time I sometimes like to create a SNOW edit. SNOW is a open world winter sports game with a pretty good replay editor built in. That way I can record the tricks I perform in the game and throw them together to make a so called 'edit'. I like doing this because I have full control over the shots, editing, music etc.
